# Импорт данных из старой БД

Эта папка содержит CSV файлы с данными из предыдущей итерации проекта.

## Формат файлов

Ожидаемые CSV файлы:
- `countries.csv` — данные стран
- `cities.csv` — данные городов
- `places.csv` — данные мест
- `events.csv` — данные событий
- `articles.csv` — данные статей
- `media_files.csv` — данные медиа файлов (для маппинга изображений)

## Использование

После добавления CSV файлов в эту папку будут созданы seed-скрипты для:
1. Импорта данных в новую БД
2. Маппинга изображений из `media_files` в соответствующие поля
3. Преобразования структуры данных под новую схему

## Примечания

- CSV файлы должны быть в UTF-8 кодировке
- Первая строка должна содержать заголовки колонок
- Пустые значения должны быть пустыми строками или `NULL`
